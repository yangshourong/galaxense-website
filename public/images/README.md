# 产品图片资源

## 图片放置说明

请将产品图片放入此文件夹，按以下命名规范：

## 必需图片

1. **product-main.png** - 产品主图（斜45度角，带测量孔视图）
   - 用于：产品页 Hero 区域
   - 建议尺寸：800x800px

2. **product-side.png** - 产品侧视图
   - 用于：产品页多角度展示
   - 建议尺寸：800x600px

3. **product-hero.png** - 首页 Hero 区域产品图
   - 用于：首页 Hero 区域
   - 建议尺寸：600x600px

## 可选图片

- **product-detail-1.png** - 产品细节（显示屏特写）
- **product-detail-2.png** - 产品细节（测量孔特写）
- **product-detail-3.png** - 产品细节（侧面接口特写）

## 图片要求

- 格式：PNG 或 WebP（推荐）
- 背景：透明或纯白背景
- 分辨率：高清（至少 2x 显示分辨率）
- 文件大小：单张图片不超过 500KB
